# 环境配置

Utopo为方便开发者调试应用，提供了两套环境，测试以及正式环境。在使用SDK的时候，配置对应的URL就可以使用对应的环境。

| 环境名 | URL地址                                   | 支持链             |
| --- | --------------------------------------- | --------------- |
| UAT | http://proxy.pingolab.com/blockchainfit | polygon testnet |
| PRD | https://www.utopo.com/blockchainfit     | polygon mainnet |
